Machine Vision-based Inspection System

The Machine Vision-based Inspection System (MVIS) is an automated system designed to detect defects in wagons. It is designed, developed, and deployed for Indian Railways in the Eastern Freight Corridor of Dedicated Freight Corridor Corporation of India Ltd (DFCCIL). Machine Vision Inspection System ( MVIS ) is innovated jointly by

  • IISc ( Indian Institute of Science)
  • IISc Startup for Railways ( Lab To Market Innovations Private Limited)
  • Indian Railways ( through DFCCIL )

This innovation is from India for INDIA and to the world and it will be a very economical solution. It has been in use for more than six months and is continuously getting upgraded.
